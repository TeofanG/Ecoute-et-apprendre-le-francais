Proiectul "Écoute et apprendre le français" - realizat de elevii Grădinariu Teofan şi Vodă Georgian este un software educaţional cu scopul pregătirii elevilor de clasa a 12-a în vederea susţinerii probei C la limba franceză a examenului de Bacalaureat. Acest proiect, sub forma unui site web este intitulat "Écoute et apprendre le français" si poate fi accesat la adresa http://nicugane.ro/franceza/
